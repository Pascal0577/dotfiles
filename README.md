# Hyprland Config
Simple dotfiles for Hyprland that I made, based on the [Catpuccin Macchiato colors](https://catppuccin.com/palette/). It uses [Hyprpanel](https://github.com/Jas-SinghFSU/HyprPanel) as a status bar.
